Exercise:

In this exercise, you will create a super hero name generator. You will use the 'Favourite colour', 'Favourite animal' and 'Favourite adjective' inputs to generate an output message with the super hero name; e.g., 'Your super hero name is The Ponderous Pink Pangolin'.

The steps will be very similar to the example above so make sure to examine the example code if you need to.

            The suggested steps in the code are:

              -  get references to required elements
              -  get the values from the inputs and store in variables
              -  calculate any new variables needed or modify existing variables
              -  update page element/s to display the result back to the user
            
            Check the JavaScript code and comments carefully to see where to write your steps.

            Optional extension steps:

              - automatically capitalise the first letter of every input
                - Remember that a user may just enter lower case words
                - You may need to research string operations
              - create a second generator with different inputs to generate something like: 
                - Rock Star Name, Super Villain Name, Disney Character Name 
                - Try to be creative and think of different inputs to the super hero name
                - You will need to create new HTML input elements in the HTML file
