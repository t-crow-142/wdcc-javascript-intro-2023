Exercise:

In this exercise, you will see that the same data from Session 2 has been included within the runExercise() function. This data consists of two arrays: languages and descriptions.

For this exercise you will reformat the data so that the two separate arrays are converted into a single array of JSObjects. Each JSObject should have a language and description property.

The structure of the array of objects would be something like this: 

            
                const languageObjectsArray = [
                    {
                        property: value,
                        property: value
                    },
                    {
                        ...
                    },
                    {
                        ...
                    }
                ];

The array of JSObjects will be similar to the array of people from the example on this page. Make sure to check the example if you're unsure of how to complete the code.

Once you have finished structuring the data, write the code required to loop through the data and display it in the HTML page. You can display it within the div with the id "displayExercise". Think carefully about how to approach this. The structure should be similar to the code in session 2 but be looping through a single array of objects with nested properties.

Optional extension task:

Add an extra property to each JSObject in the array of objects. This extra property should be called "features" and it should be an array that contains a list of features for each language. Add some features into the "features" array for each of the three languages. It can be a varying amount of features for each language.

Now write code that will automatically display all data from the array of languages objects in the page including the nested array. It should show the items within the nested arrays as unordered lists that display below the language name and description for each language.
