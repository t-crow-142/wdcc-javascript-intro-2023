General information:

All the instructions for the exercises in these sessions are included in the HTML file within which you will also run the exercises and examples.

You will see that each individual page will have a matching JavaScript file within which you can write your code for each exercise.

You can edit the code for the files with any code editor; however, it is recommended that you use Visual Studio Code as this is what will be used in the live demonstrations.

We will only be aiming to cover the content for 3 of the sessions today; however, there is extra content in the slides and extra exercises available for those who want to spend more time on JavaScript after the workshop.

You can browse between the examples and exercises for each of the four sessions with the navigation links above.
